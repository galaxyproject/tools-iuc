# Trackastra model data manager

Installs Trackastra model directories into the `trackastra_models` Galaxy Tool Data Table. It supports the versioned upstream `general_2d` and `ctc` releases and administrator-supplied model ZIP archives.

## Upgrade note

Version `0.1.1` removes shell-continuation backslashes from rendered Galaxy
commands. Replace version `0.1.0` when testing the model data manager.
