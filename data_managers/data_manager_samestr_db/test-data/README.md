# Test Data for SameStr Data Manager

## MetaPhlAn Test Database

The file 'mpa_test-db-without-one-marker.pkl' was generated from 'mpa_test-db-without-one-marker.json' using the 'customizemetadata.py' script.

Both original files can be found in the MetaPhlAn tool repository:
https://github.com/galaxyproject/tools-iuc/tree/main/tools/metaphlan

### Regenerating the PKL file

If needed, the PKL file can be generated by using the following command:

```bash
python customizemetadata.py transform_json_to_pkl \
    --json mpa_test-db-without-one-marker.json \
    --pkl mpa_test-db-without-one-marker.pkl
```
